# 常见问题

Mac下已经安装了Python3，但是到找不到python3命令。解决方法：建立软连接

```shell
ln -s /usr/local/Cellar/python/3.6.4_2/bin/python3 /usr/local/bin/python3
```

Pycharm中配置Python版本号

File->Settings->Project Interpreter
